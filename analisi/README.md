# Analisi

Cartelle ordinate per argomento. I notebook di analisi stanno dentro `analisi/<categoria>/notebooks/`.

Regola fonti per le analisi attive: usare solo API ufficiali. Gli script non scrivono CSV; le tabelle restano in memoria e i grafici PNG vengono salvati nella stessa cartella del codice che li produce.

- `cruscotto/`: vista trasversale del declino italiano.
- `pil/`: PIL, PIL pro capite, crescita reale e confronti internazionali.
- `valore_aggiunto/`: valore aggiunto, ore lavorate e produttivita settoriale.
- `occupazione/`: occupati, tassi di occupazione, territorio, istruzione, sesso e settore.
- `demografia/`: popolazione, natalita, fertilita, invecchiamento e dipendenze.
- `istruzione/`: livelli di istruzione, partecipazione, occupazione per titolo di studio.
- `ricerca/`: R&S, brevetti, innovazione e capitale tecnologico.
- `istituzioni/`: burocrazia, governance e qualita regolatoria.

Cartelle tecniche:

- `utils/`: funzioni condivise per indicatori API trasversali.

# Analisi

Cartelle ordinate per argomento. Ogni cartella contiene note operative e puo ospitare notebook, script e output dedicati.

- `pil/`: PIL, PIL pro capite, crescita reale e confronti internazionali.
- `valore_aggiunto/`: valore aggiunto, ore lavorate e produttivita settoriale.
- `occupazione/`: occupati, tassi di occupazione, territorio, istruzione, sesso e settore.
- `demografia/`: popolazione, natalita, fertilita, invecchiamento e dipendenze.
- `istruzione/`: livelli di istruzione, partecipazione, occupazione per titolo di studio.
- `ricerca/`: R&S, brevetti, innovazione e capitale tecnologico.

Cartelle tecniche:

- `_comune/`: funzioni condivise e output degli indicatori API trasversali.
